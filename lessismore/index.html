About LessIsMore

LessIsMore is a simple, light-weight CSS Framework built using the Less preprocessor. I'm proud to say I have used ZERO Javascript or jQuery to develop this framework, and will continue to develop it without the use of Javascript or jQuery.

I wanted to make a functional, light-weight framework that didn't rely on Javascript or jQuery, and is able to do the same things a framework using either can do. I also wanted my framework to improve DOM loading by eliminating the need for Javascript or jQuery plugins which slow down how fast your page loads. I will include OPTIONAL Javacript form validation in a separate section if there is interest.

Grid System

Here is the simple grid system that LessIsMore ships with. The background gradient is simply there to show how much of your screen is taken up by a column of each size and will NOT be included in the download of this framework. Don't want all of the extra add ons? Feel free to simply download the grid system. Want to make LessIsMore better? Feel free to contribute!
